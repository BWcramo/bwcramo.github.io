# BWcramo — Site GitHub Pages initial

Ce dépôt sert à héberger la page statique initiale du projet BWcramo via GitHub Pages (https://bwcramo.github.io).

Contenu fourni :
- `index.html` : page d'accueil simple (prototype).
- `.nojekyll` : empêche GitHub Pages d'utiliser Jekyll (utile si tu as des dossiers `_`).
- `.gitignore` : fichier de base.
- `LICENSE` : licence MIT.

Étapes rapides pour déployer (résumé) :
1. Créer l'organisation `bwcramo` sur GitHub (ou utiliser un compte `bwcramo`).
2. Créer un repo nommé exactement `bwcramo.github.io` sous l'organisation.
3. Cloner / initialiser ce dossier localement, copier les fichiers fournis.
4. Pousser sur la branche `main`.
5. GitHub Pages servira le site à l'adresse https://bwcramo.github.io

Je te guide étape par étape dans la suite du README (SSH, push, vérification).
